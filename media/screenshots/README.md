# Screenshot capture guide

Drop PNGs here with these exact filenames so the README renders them:

| File | What to capture |
|------|-----------------|
| `chat-streaming.png` | New chat UI with the agent picker, workspace attachments, streamed response, and tool activity visible. |
| `create-agent.png` | Create-agent dialog showing name, role, starting point, and the primary Create agent action. |
| `agent-settings.png` | In-chat Agent settings page showing the profile card and model connection fields. Never expose a real API key. |

## Tips

- Use a clean VS Code window and the same theme for all screenshots.
- Crop to the sidebar + a bit of editor context, no need to capture the full screen.
- Keep file size under ~300 KB each (use `pngquant` or similar) so the marketplace listing loads fast.
- For an animated GIF, also drop one as `chat-streaming.gif` and update the README image to point at it.
